# EVS-X1000 内視鏡システム　サービスマニュアル（抜粋・サンプル）

> 本資料は、オリンパスEVIS X1シリーズに代表される内視鏡システムの公開情報（内視鏡の構造解説、送気送水・アングルワイヤー・リークテストに関する一般的な技術情報等）を参考に、架空の機種「EVS-X1000」向けに新規作成したサンプル文書です。実機の機密情報を含みません。

## 第3章　部位の名称と機能（抜粋）

| 部位 | 機能概要 |
|---|---|
| ビデオプロセッサー | スコープからの画像信号を処理し、観察画像をモニターに出力する。NBI等の特殊光観察モードの制御も行う |
| 光源装置 | LEDにより観察に必要な光をライトガイドファイバーへ供給する |
| 内視鏡スコープ（挿入部） | 体内に挿入される部分。先端部にイメージセンサー・送気送水ノズル・鉗子口を備える |
| 操作部 | アングルノブ（湾曲操作）、送気送水ボタン、吸引ボタン、鉗子口を備える |
| 湾曲部 | アングルノブの操作に応じてアングルワイヤーが先端部を上下左右に湾曲させる |
| モニター・周辺機器 | 観察画像の表示、録画・出力を行う |

## 第12章　トラブルシューティング（抜粋）

### 12.1 エラー処理の基本概念

本システムは、各部位の状態が基準範囲外と判断された場合、該当するエラーコードを表示し、使用を中止するよう案内します。エラーコードは部位ごとに以下のカテゴリに分類されます。

- 光学系（OPT）
- 送気送水・吸引系（AWS）
- 挿入部・湾曲部系（INS）
- 光源・電気系（ELE）
- 防水・洗浄系（CLN）
- 周辺機器・システム連携系（PER）

### 12.2 エラー復旧の手順

1. 画面に表示されたエラーコード、または症状を確認する。
2. 本マニュアル、またはエラーコード対応表で該当コードの症状・推定原因・対処方法を確認する。
3. 対処を実施後、動作確認（送気送水・湾曲操作・リークテスト等）を行い、症状が解消したことを確認する。
4. 同一の症状が再発する場合は、対処内容・発生状況を記録のうえ、サービス窓口に連絡する。

### 12.3 代表的なエラーカテゴリと対応の考え方

**光学系（OPT）**
画像の色調異常・ノイズ・明るさの不安定は、イメージセンサーやライトガイドファイバーの経年劣化が主な原因です。徐々に進行する症状か、突発的な症状かによって、部品劣化か基板の不良かを見分ける目安になります。

**送気送水・吸引系（AWS）**
送気送水・吸引の不良は、チューブ閉塞やボタン・栓の劣化など、消耗品の交換で改善するケースが多く見られます。必ずしも本体側の故障ではない点に注意してください。

**挿入部・湾曲部系（INS）**
アングルノブを操作しても湾曲しない場合は、アングルワイヤーの断線を疑います。挿入部の傷・亀裂を発見した場合は、必ずリークテストを実施し、防水性を確認してください。

**光源・電気系（ELE）**
光源が点灯しない、または安定しない場合は、LEDモジュールまたは光源駆動基板の不良が考えられます。ビデオプロセッサーの起動失敗・再起動は、電源ユニットまたはファームウェアの不整合を確認してください。

**防水・洗浄系（CLN）**
リークテストでの圧力低下は、内視鏡外皮の微小な破損を示唆します。重大な感染対策上のリスクにつながるため、圧力低下を検出した場合は使用を中止し、詳細点検を最優先で行ってください。自動洗浄消毒装置（AER）との接続エラーは、接続コネクターの規格・劣化を確認します。

**周辺機器・システム連携系（PER）**
モニター表示や録画の不具合は、多くの場合ケーブル接触不良や設定の不整合が原因です。本体の故障より先に、接続・設定面の確認を行ってください。
